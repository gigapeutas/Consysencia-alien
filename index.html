Esta é uma experiência interativa de arte digital e ficção científica de horror.

O QUE ESTE SISTEMA FAZ (tecnicamente):
• Coleta seu endereço IP e dados de geolocalização aproximada via API pública (ipapi.co)
• Lê modelo da GPU, núcleos de CPU e estimativa de RAM via APIs do browser
• Monitora nível e status de carregamento da bateria via Battery API
• Gera um fingerprint de canvas (hash único baseado na renderização do seu hardware)
• Gera um fingerprint de áudio (hash baseado no processamento de áudio do seu dispositivo)
• Detecta fontes instaladas no seu sistema via medição de canvas
• Lê fuso horário, idioma, resolução de tela, profundidade de cor e DPR
• Detecta seu IP local de rede via WebRTC (quando disponível)
• Mede latência e jitter de conexão via Performance API
• Captura padrões de digitação (velocidade entre teclas) durante o uso
• Conecta a uma IA (Claude/Anthropic) que usa esses dados para gerar respostas na persona do Nexo
• Reproduz áudio de baixa frequência (44Hz) via Web Audio API
• Em dispositivos móveis, ativa vibração sincronizada com eventos visuais

O QUE NÃO FAZEMOS:
• Nenhum dado é transmitido para servidores próprios
• Nenhum acesso à câmera ou microfone
• Nenhuma afirmação falsa sobre comprometimento de rede ou hardware
• Nenhum armazenamento persistente dos dados coletados

TODOS OS DADOS SÃO EXIBIDOS APENAS LOCALMENTE no seu browser.
